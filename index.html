Yet Another Static Site (Generator)

As name says, it is static site generator written in Ada. It is headless application (no user interface, use it from console). The program is available only for Linux 64-bit. For more informations about the program, please look at documentation.

Quick start

Features

  • Support almost infinite amount of custom tags in HTML templates (depends on available RAM)
  • Separated tags for whole site and each page
  • Fast
  • Can be extended with modules, written in any script/programming language
  • Freshly created, unpolished, have a few bugs :D
